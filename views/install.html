You need to install Lune (regular version), get set up with Luau LSP, and `git submodule add` lynx from its repo to get started.

Dependencies

  • Lune Runtime

    The Lune runtime allows you to run Luau as a general purpose programming language, giving you access to filesystem, net, process (executing applications/shell commands), and other apis

    • Lune
    • Lune docs

  • Luau Language Server (LSP)

    Gives you VSCode syntax highlighting for Luau, autocomplete, type information, etc. You should be able to install it from the VSCode Extensions marketplace

    • Luau LSP

  • Lynx

    Minimal webapp wrapper for Luau/Lune

    • Lynx docs
